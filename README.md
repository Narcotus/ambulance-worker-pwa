# СМП Ассистент — справочник фельдшера

PWA для работников скорой медицинской помощи: приказы, клинические рекомендации, медицинские калькуляторы и шпаргалки.

## Запуск

```bash
pnpm install
pnpm dev
```

## Сборка

```
pnpm build
# статические файлы в out/
```

## Деплой на GitHub Pages

1. Создайте репозиторий `ambulance-worker-pwa`
2. Залейте проект
3. Settings → Pages → Source: GitHub Actions
4. Пуш в main — сайт соберётся автоматически

## APK

1. Деплойте на GitHub Pages
2. Откройте [https://www.pwabuilder.com](https://www.pwabuilder.com)
3. Вставьте URL сайта
4. Скачайте APK
